== Description ==
このプラグインをインストールすると、投稿画面のカテゴリーボックスの以下の動作を調節することができます。

1. チェックボックスが移動するのを停止する
2. チェックボックスをラジオボタンにする
3. 親カテゴリーを選択できなくする


== Installation ==

1. `/wp-content/plugins/` ディレクトリに `adjust-admin-categories` ディレクトリを作成し、その中にプラグインファイルを格納してください。
一般的には .zip から展開された check-copy-contentsフォルダをそのままアップロードすれば OK です。
2. WordPress の "プラグイン" メニューから "Adjust Admin Categories" を有効化してください。
3. Adjust Admin Categoriesのオプション設定は "設定 > Adjust Admin Categories" で行えます。
